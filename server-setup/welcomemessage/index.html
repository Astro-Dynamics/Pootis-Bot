    Welcome & Goodbye Message

    Pootis-Bot can have a custom welcome and goodbye message.

    Setting welcome message

    The welcome message is a message that is sent to a specific channel everytime a new user joins the server.

    To set the welcome message use the command:

    setupwelcomemessage [message]

    You would set the [message] part to your message you want to greet new users with.

    Example:

    setupwelcomemessage Hello and welcome!

    You can also use [user] and [server] in the message to mention the user and put in the current server name.

    Example:

    setupwelcomemessage Hello [user]! Thanks for joining [server]! Please check out the rules first then enjoy your stay

    Setting goodbye message

    The goodbye message is basically exactly the same as the welcome message, except it is sent when a user leaves.

    To set the goodbye message use the command:

    setupgoodbyemessage [message]

    Example:

    setupgoodbyemessage Goodbye!

    Once again there is [user] but there is NO [server]

    Example:

    setupgoodbyemessage Goodbye [user]. We hope you enjoyed your stay.

    This would produce something that looks like this:

    Setting the channel

    Once you have your messages set its now time to set the channel they will be sent to!

    The base command for this is:

    togglewelcomemessage [channel]

    If this is your first time setting a welcome channel then you have to input a name.

    Example:

    togglewelcomemessage welcome

    This would enable the welcome and goodbye messages and set it to the channel #welcome

    Disabling it would also be easy, just re-run the same command but without a channel name.

    Enabling goodbye message

    By default the goodbye message is disabled, you can enable it using this command:

    togglegoodbyemessage

    You can run the same command to disable it.

    Note

    It will use the same channel setup for the welcome message!

    This means you need the welcome message setup and set to a channel!
