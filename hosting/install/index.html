    Installation

    .NET Core

    If you already have .NET Core 3.0 installed, you can skip this part.

    Download

    First download .NET Core 3.0. If all you want to do is run the bot (no development), then all you need to install is the .NET Core Runtime, not the SDK.

    Install

    Its pretty simple to install, just like a normal Windows installer, or if you are on a Unix system, follow the instructions on the .NET website.

    Pootis-Bot

    Download

    First off, start by downloading the latest version of Pootis-Bot for your OS.

    The downloaded file should look like pootis-bot-[os]-[architecture]-[version].zip. So for a Windows 64 bit version of Pootis-Bot, it should look like pootis-bot-win-x64-1.0.0.zip. NOTE: The version number will change depending on how many releases have been made since this was written.

    Once you have downloaded it, extract the downloaded .zip file.

    Install

    Pootis-Bot is essentially a portable app, though you want to place the Pootis-Bot files some were you can access easily, and preferable some were without admin permissions.

    If you are on Windows you can start the bot via the Pootis-Bot.exe.

    If you want to run it from the console, or you are not on Windows, then open a terminal to were the files are located and run the command dotnet Pootis-Bot.dll.
