    Config File

    Here is a overview of the config file.

    This is for config version: 2, Pootis-Bot version: 0.2.6. Updated 22/07/19

    Default config

    {
  "configVersion": "2",
  "botToken": "",
  "botPrefix": "$",
  "botName": "Bot",
  "twichStreamingSite": "https://www.twitch.tv/creepysin",
  "gameMessage": "Use $help for help.",
  "isAudioServiceEnabled": false,
  "checkConnectionStatus": true,
  "checkConnectionStatusInterval": 60000,
  "apis": {
    "apiGiphyKey": null,
    "apiYoutubeKey": null,
    "apiGoogleSearchKey": null,
    "googleSearchEngineID": null
  },
  "helpModules": [
    {
      "group": "Basic",
      "modules": [
        "BasicCommands",
        "Misc"
      ]
    },
    {
      "group": "Utils",
      "modules": [
        "Utils"
      ]
    },
    {
      "group": "Fun",
      "modules": [
        "GiphySearch",
        "GoogleSearch",
        "YoutubeSearch",
        "TronaldDump"
      ]
    },
    {
      "group": "Audio",
      "modules": [
        "Music"
      ]
    }
  ]
}

    Notice

    A lot of these options are in the bot's config menu, If you want to access the bot's config menu, run the command config in the console.

    What does each key mean?

    General Options

    configVersion - The config's version, its best to leave this alone since it is only used if the bot's config is updated and it needs to be saved again.

    botToken - The bot's token, you can set this in the config menu.

    botPrefix - What prefix to use, you can set this in the config menu.

    botName - The bot's name, you can set this in the config menu.

    twichStreamingSite - If bot is in streaming mode, what Twitch site should it show, by default it set to mine.

    gameMessage - Default game message, in case the bot restarts it will default to this.

    isAudioServiceEnabled - Are audio services enabled.

    checkConnectionStatus - Checks connection status every so often.

    checkConnectionStatusInterval - How often to check the connection status, in milliseconds.

    API Keys

    All of these can be set in the config menu under apis.

    apiGiphyKey - Giphy api key.

    apiYoutubeKey - YouTube api key.

    apiGoogleSearchKey - Google Search api key.

    googleSearchEngineID - Google Search Engine ID.

    Help Modules

    Each help module is structured like this:

    {
  "group": "[GROUP NAME]",
  "modules": [
    "[MODULE NAME]"
  ]
}

    Group - Refers to the name of group in each help section modules - Has a list of all the modules you want to include in that section.
