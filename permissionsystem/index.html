Permission System

Pootis Bot includes a permission system.

Setting a permission

Set a permission for a command using these command:

perm [Command] [Role]

The command must be the exact spelling of the command and same goes for the role, exempt that must be a role that exist on your server.

EG: perm youtube Admin

Removing a permission

Removing a permission is simple, type:

perm [Command] Remove

This will remove the permission.
